Statistical Guarantees in Data-Driven Nonlinear Control: Conformal Robustness for Stability and Safety

Ting-Wei Hsu & Hiroyasu Tsukamoto
Department of Aerospace Engineering
University of Illinois Urbana-Champaign
Paper arXiv Code

Abstract

We present a true-dynamics-agnostic, statistically rigorous framework for establishing exponential stability and safety guarantees of closed-loop, data-driven nonlinear control. In particular, we consider a nonlinear dynamical system given by \begin{align} \label{eq:true_dyn} \dot{x}=f(x, u), \end{align} where $x$ $x$ is the state vector, $u$ is the control input vector, and $f:\mathcal{X} \times \Ucal \rightarrow \mathcal{X}$ is an unknown locally Lipschitz function.

Central to our approach is the novel concept of conformal robustness, which robustifies the Lyapunov and zeroing barrier certificates of data-driven dynamical systems against model prediction uncertainties using conformal prediction. It quantifies these uncertainties by leveraging rank statistics of prediction scores over system trajectories, without assuming any specific underlying structure of the prediction model or distribution of the uncertainties.

With the quantified uncertainty information, we further construct the conformally robust control Lyapunov function (CR-CLF) and control barrier function (CR-CBF), data-driven counterparts of the CLF and CBF, for fully data-driven control with statistical guarantees of finite-horizon exponential stability and safety. The performance of the proposed concept is validated in numerical simulations with four benchmark nonlinear control problems.

Conformal Prediction

Suppose we are given a dataset $S:= \left\{(X_i, Y_i) \right\}_{i\in\mathcal{I}}$, where $X_i$ and $Y_i$ are the features and labels and $N:= |\mathcal{I}|$. Given any regression model $\hat{Y}_i = \hat{\mathcal{F}}(X_i)$, we define the nonconformity scores as \begin{align} s_i := \|\hat{\mathcal{F}}(X_i)-Y_i\|,~\forall i\in \mathcal{I}.\in \end{align} Intuitively, smaller scores imply a more accurate regression model. Given a failure probability $\delta\in(0,1)$, we define the conformal quantile as the $(1-\delta)$th quantile of the empirical distribution of $\{s_i\}_{i\in\mathcal{I}}\cup\infty$, denoted by \begin{align} \label{eq:quantile} q_{1-\delta} := \mathrm{Quantile}\Big(1-\delta; \sum_{i=1}^{N} \tfrac{1}{1+N} \delta_{s_i} + \tfrac{1}{1+N} \delta_{\infty}\Big),~~ \end{align} in which $\delta_a$ denotes the point mass probability at $a\in\mathbb{R}\cup\{-\infty, \infty\}$. This quantile can be equivalently interpreted as the $\lceil (1-\delta)(N+1)\rceil$-th smallest nonconformity score in $\{s_i\}_{i\in\mathcal{I}}$. $\lceil\cdot\rceil$ denotes the ceiling function. If, for any given new sample $(X_{i_{\mathrm{new}}}, Y_{i_{\mathrm{new}}})$, all the nonconformity scores in $\{s_i\}_{i\in\mathcal{I}\cup i_{\mathrm{new}}}$ are exchangeable, then we have the following statistical bound on the prediction error: \begin{align} \mathrm{Pr}\{\|\hat{f}(X_{i_{\mathrm{new}}})-Y_{i_{\mathrm{new}}}\| \leq q_{1-\delta} ~\vert~ S \} \geq 1-\delta. \end{align}

Conformally Robust Control Lyapunov Function (CR-CLF)

Suppose we are given a data-driven model $\hat{f}(x,u)$ of an unknown dynamical system and a conformal quantile $\quantile$ defined in \eqref{eq:quantile}. A $C^1$ function $V:\mathcal{X}\rightarrow\mathbb{R}$ is a conformally robust CLF (CR-CLF) if it satisfies \eqref{clf_cond_1}, and $\forall x\in\mathcal{X}$, $\exists u\in\Ucal$ s.t. \begin{align} \label{eq:crclf_cond} \tfrac{\partial V}{\partial x}\hat{f}(x,u) + c_3 V(x) + \left\|\tfrac{\partial V}{\partial x} \right\| \cdot \quantile \leq 0. \end{align} \end{definition} % \begin{theorem} \label{th:crclf_policy} Consider a set of control inputs given by $\mathbf{K_{CL}}(x) = \big\{u\in\Ucal \big\vert \tfrac{\partial V}{\partial x}\hat{f}(x,u) + c_3 V(x) +\|\tfrac{\partial V}{\partial x}\| \quantile \leq 0 \big\}$. Any locally Lipschitz continuous control policy $u(x)\in \mathbf{K_{CL}}(x)$ renders $\Prob\{ V(x_t) \leq V(x_0) e^{-c_3 t}, \forall t\in[0,T]\} \geq 1-\delta$ and $\Prob\{\|x_t\| \leq \sqrt{c_2/c_1}\|x_0\| e^{-c_3 t/2}, \forall t\in[0,T]\} \geq 1-\delta$, for all $x_0 \in \mathcal{D}$. \end{theorem}
